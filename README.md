# Maa Laxmi Fish Suppliers - Accounting System

A modern, high-performance accounting and ledger management system designed for fish supply businesses. This application provides real-time financial tracking, company management, and automated reporting.

## 🚀 Key Features

- **Dynamic Dashboard**: Real-time analytics with interactive charts (Sales Trends, Asset Allocation, and Live Ledger Pulse).
- **Company Management**: Complete directory for clients and suppliers with automated balance calculations.
- **Transaction Tracking**:
  - **Add Sale**: Record sales with detailed weighting (boxes, weight per box, rate).
  - **Add Payment**: Track Cash/Bank payments and Custom adjustments (Penalties/Fees).
  - **Add Received**: Manage inventory and procurement records.
- **Professional Reporting**:
  - Comprehensive Statement of Account (Ledger) for every company.
  - Modern and Classic view modes.
  - Export functionality to optimized PDF and high-resolution PNG.
- **Localization**: Native support for the Nepali Date System (BS).

## 🛠️ Technology Stack

- **Frontend**: React (Vite) + Framer Motion (Animations)
- **Styling**: Vanilla CSS (Premium Modern UI)
- **Database**: Supabase (Real-time Backend-as-a-Service)
- **Analytics**: Recharts (Dynamic Financial Graphs)
- **Icons**: React Icons (Io5, Fi, Hi2)

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables in `.env`:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   VITE_DELETE_PASSCODE=your_passcode
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
